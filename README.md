# ERP Lead Intelligence Engine

An AI-powered platform that automates ERP lead research, business analysis, opportunity identification, and sales outreach.

## Project Goals

- Research companies automatically
- Analyze ERP opportunities
- Recommend Inoday solutions
- Generate outreach messages
- Export structured lead reports

## Tech Stack

- Python
- Pydantic
- Playwright
- OpenAI / GPT
- Git
- VS Code
